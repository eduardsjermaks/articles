- The container chmods many writable runtime dirs (`App_Data`, `Plugins`, `wwwroot/bundles`, `wwwroot/images/uploaded`, etc.) — non-Docker hosts must ensure these are writable ([Dockerfile:21-33](Dockerfile#L21-L33)).
